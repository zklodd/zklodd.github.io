Computer Architecture

Computer Architecture was a class that was focused around the base framework and logic of a computer and how it properly "talks" to a user. Using MIPS assembly code, it will translate given instrucitons into a machine readable format and performs actions from there. 

In this project, I needed to mock the way a reduced computer would interact like this using Java. To make this work I needed to take two files, one that represents the first portion of a MIPS file, the .text section, and one that represents the second portion of the MIPS file, the .data section. The program will then read these files, which will be in MIPS instruction format, then translate it into the machine readable format, taking the logic of each instruction and executing it as expected. The PDF file attached shows an example run of a given MIPS program, as well as the source code.

The zip file attached contains the course submission and everything that was needed to run the sample execution in the PDF file.
