Introduction to Software Engineering

Introduction to Software Engineering, or COMPSCI 361, was a class I took revolving around full stack development aspects of computer science. This class focused on real life circumstances, methodologies to be used, and introduced a new framework to me, Django.

In this class, I learned how to navigate around the Django framework, using Python, web development like HTML, CSS, and JavaScript, and SqlLite, to create a full-fledge application. I worked in a team of 4 using Agile environments, it invovled weekly scrum meetings, activity boards, and testing/documentation of our work. This semester long project showed me what work in the real world will look like, and prepared me for professional demonstration of my software development abilities, it taught me that not only do I need to program well, but I also need to translate to a team the work I've done and how it improves our business.

For this project, we needed to develop a Github Repository that will create a mock "Event Management System" for UWM, from here there will be multiple users with different roles and permissions when it comes to accessing the website and its resources, for example, an organizer has the ability to create events with dates and locations that students may sign up and register for, as well as oversee the number of students already regsitered.

The part of the project that I focused on was the administrator's page, an administrator has the ability to manage users, event locations and subjects, and approve or deny any event requests submitted by an organizer. A link to the github repository has been attached for visibility
