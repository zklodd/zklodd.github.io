American Champion Aircraft

American Champion Aircraft is a Airplane Fabrication and Repair company. It is also the company my dad is the manager of. He was requesting that I would write a program for him that determines what parts that customers would order would be overdue, in other words, when they need to fabricate a part for an airplane, they would be placed into a list in google sheets, and then be assigned a date it was started, and the date it is due. The program I made develops a two pie charts based on the entries provided. The first one shows how many parts are past their due date, and the second one shows what percentage of departments hold the overdue parts. 

This program uses the Google, "Apps Scripts," which is a cloud-based, JavaScript platform powered by Google.

The PDF file attached shows an example input with a list of parts and due dates assigned, as well as each of the pie charts from the given data. Lastly it shows the source code of each function that is used to generate the pie charts and insert it into the sheet.
